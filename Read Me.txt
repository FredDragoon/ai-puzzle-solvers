There isn't much required for this particular project to get it up and running
all the files are provided in the project folder and even then most of those weren't necessarily require like the puzzles.py 
but I found it helpful for modularity in case the puzzles in the future could/needed to be swapped out/make the main.py cleaner.

The only library was imported was "time" and that is part of the python native library so no further things need to be installed.

The user just needs to run main.py and the program should output the solved sudoku's of each difficulty showing they were solved correctly 
and the trials/time it took to solve each Backtracking method(simple vs smart)

(python intepreter may vary depending on the version of python a user is using)
